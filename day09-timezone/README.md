# Conversor de Zonas Horarias 🌐

Convertí cualquier hora a 12 ciudades del mundo al instante.

## 🚀 Demo
[Ver en vivo](https://lucaskraglievich.github.io/day09-timezone)

## ✨ Features
- 12 ciudades principales del mundo
- Muestra hora actual por defecto
- Zona origen resaltada visualmente
- Sin librerías externas — usa Intl.DateTimeFormat nativo del browser

## 🛠 Stack
HTML · CSS · JavaScript · Intl API

## 💡 Conceptos aplicados
- `Intl.DateTimeFormat` para conversión de zonas horarias sin librerías
- Zonas horarias IANA (el estándar internacional)
- Cálculo de offsets UTC para convertir correctamente entre zonas

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor ni API key.

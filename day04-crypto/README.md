# Crypto Tracker 📈

Top 20 criptomonedas en tiempo real con precios y variación en 24hs.

## 🚀 Demo
[Ver en vivo](https://lucaskraglievich.github.io/day04-crypto)

## ✨ Features
- Top 20 por capitalización de mercado
- Variación 24hs en verde/rojo
- Filtro en tiempo real sin llamadas extra a la API
- Sin API key necesaria

## 🛠 Stack
HTML · CSS · JavaScript · CoinGecko API

## 💡 Filtro en memoria
Los datos se guardan en una variable al cargar. Al filtrar, se busca
sobre esos datos sin volver a llamar a la API — más eficiente y más rápido.

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor.

# Prode Mundial 2026 🇦🇷

Predecí los resultados de Argentina en la fase de grupos del Mundial 2026.

## ✨ Features
- Predicciones para los 3 partidos de grupo de Argentina
- Persistencia con LocalStorage — tus predicciones quedan guardadas al recargar
- Resumen visual de todas las predicciones guardadas
- Opción de borrar todo y empezar de nuevo

## 🛠 Stack
HTML · CSS · JavaScript

## 💡 Conceptos aplicados
- `localStorage.setItem()` / `getItem()` para persistencia de datos
- `JSON.stringify()` / `JSON.parse()` para guardar objetos como texto
- Generación dinámica de HTML con template literals
- Manejo de estado simple sin frameworks

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor ni API key.

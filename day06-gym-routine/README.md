# Generador de Rutina de Gym 💪

Generá tu rutina al instante según grupo muscular y nivel.

## 🚀 Demo
[Ver en vivo](https://lucaskraglievich.github.io/day06-gym-routine)

## ✨ Features
- 6 grupos musculares: pecho, espalda, piernas, hombros, bíceps, tríceps
- 3 niveles: principiante, intermedio, avanzado
- Series y repeticiones para cada ejercicio
- Copiá la rutina completa al portapapeles

## 🛠 Stack
HTML · CSS · JavaScript

## 💡 Conceptos aplicados
- Objeto anidado como base de datos local (músculo → nivel → ejercicios)
- Manejo de estado de selección activa con clases CSS
- Generación dinámica de HTML a partir de datos estructurados
- Clipboard API para copiar texto al portapapeles

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor ni API key.

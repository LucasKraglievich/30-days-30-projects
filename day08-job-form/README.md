# Formulario de Postulación Laboral 📋

Formulario con validación en tiempo real y feedback visual instantáneo.

## ✨ Features
- Validación mientras escribís, no solo al enviar
- Regex para email y URL de LinkedIn
- Borde verde/rojo + mensaje de error específico por campo
- Botón de envío deshabilitado hasta que todo sea válido
- Validación de rango numérico para años de experiencia

## 🛠 Stack
HTML · CSS · JavaScript

## 💡 Conceptos aplicados
- Expresiones regulares (regex) para email y URL
- `event.preventDefault()` para evitar el recargo de página
- Objeto de estado para trackear la validez de cada campo
- Validación condicional con feedback específico por tipo de error

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor ni API key.

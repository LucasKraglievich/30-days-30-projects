# 30 Days 30 Projects 🤖

Un proyecto nuevo cada día durante 30 días, como ejercicio de práctica y portfolio.

## Sobre el challenge

Arranqué con proyectos en HTML, CSS y JavaScript puro, consumiendo APIs externas y aplicando distintos conceptos del lenguaje. La idea es ir subiendo la complejidad día a día, avanzando más adelante hacia Node.js y Java con Spring Boot.

Cada carpeta tiene su propio README con el detalle del proyecto, el stack usado y el link a la demo en vivo.

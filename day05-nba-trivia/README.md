# Trivia NBA 🏀

Quiz interactivo sobre la Final NBA 2026 y la historia de la liga.

## ✨ Features
- 10 preguntas sobre la Final 2026 (Spurs vs Knicks) e historia NBA
- Timer de 15 segundos por pregunta con barra animada
- Feedback inmediato con dato extra tras cada respuesta
- Preguntas en orden aleatorio en cada partida
- Resumen final con puntaje y detalle de respuestas

## 🛠 Stack
HTML · CSS · JavaScript

## 💡 Conceptos aplicados
- `setInterval` para el countdown del timer
- `clearInterval` para limpiar el timer al responder
- Fisher-Yates shuffle para mezclar las preguntas aleatoriamente
- Manejo de estados (inicio, juego, resultado) con display none/block

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor ni API key.

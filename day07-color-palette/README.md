# Generador de Paletas de Colores 🎨

Generá paletas de 5 colores al instante. Bloqueá los que te gustan y regenerá el resto.

## ✨ Features
- Paleta de 5 colores aleatorios
- Hover para ver el HEX y las acciones
- Bloqueá colores para que no cambien al regenerar
- Copiá el código HEX con un clic
- Barra espaciadora para generar nueva paleta
- Texto blanco o negro automático según luminancia del fondo

## 🛠 Stack
HTML · CSS · JavaScript

## 💡 Conceptos aplicados
- Cálculo de luminancia WCAG para contraste de texto
- Conversión de decimal a HEX con toString(16) y padStart
- Event listener global en el teclado (spacebar)
- Estado mutable con array de objetos

## 📌 Cómo usarlo
Abrí `index.html` en el navegador. No necesita servidor ni API key.
